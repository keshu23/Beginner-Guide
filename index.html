- A formal computer language with a controlled vocabulary and grammatical rules designed to instruct a computer how to perform specific tasks.

- Programming languages create programs to control machine behavior or express algorithms.

- Used to develop desktop applications, websites, and mobile applications.

Low Level Languages

- Machine-dependent (0s and 1s) programming language.

- Runs directly without a compiler or interpreter, making programs very fast.

Examples: Machine language, Assembly language.

High Level Languages

- Designed for user-friendly software and websites.

- Requires a compiler or interpreter to translate into machine language.

- Easy to read, write, and maintain.

Examples: Python, Java, JavaScript, PHP, C#, C++, Objective C, Cobol, Perl, Pascal, LISP, FORTRAN, Swift.

Divided into:

  • Procedural Oriented Programming
  • Object-Oriented Programming

Middle Level Programming Languages

- Lies between low-level and high-level languages.

- Supports high-level features, user-friendly, and close to machine and human language.

Examples: C, C++.

Importance of ReactJS
